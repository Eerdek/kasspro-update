# KassPro Update

KassPro POS системийн авто-шинэчлэлтийн manifest.

`latest.json` — гарын үсэгт шинэчлэлтийн мэдээлэл (version, url, sha256, notes).
Апп интернэттэй үедээ татаж, гарын үсэг + SHA-256-г батлаад л шинэчилдэг.
Хуурамч update түлхэх боломжгүй (зөвхөн эзний private түлхүүрээр зурсан нь суудаг).

Шинэ хувилбар гаргах: KassPro.LicenseGen → "Программын ШИНЭЧЛЭЛТ (latest.json) үүсгэх"
→ Setup.exe-г GitHub Release дээр тавь → гарсан latest.json-оор энэ файлыг соль.
